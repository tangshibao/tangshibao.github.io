注册GitHub构建静态博客

  1. 注册GitHub账号，并验证邮箱。

    这一步需要验证邮箱，否则不能使用gh-pages

  2. 申请一个新的仓库，用来存放该项目

    该仓库的名字需命名为[yourname].github.io。比如我的用户名为tangshibao，此处的仓库名应该写做

    tangshibao.github.io

  1. 设置gh-pages

    在该仓库中点击setting，

下拉到设置gh-pages的地方（GitHub Pages设置区）

点击Launch automatic page generator，挑选中意的主页（暂时，因为随后会使用Hexo生成静态博客替换掉该页面），Github将会自动替你创建出一个gh-pages的页面。

配置Git中个人信息

对于GitHub，我们使用工具Git来提交，下载项目。这时需要配置GitHub的信息在Git中，Git提供了Git Bash工具来操作命令

  1. 配置用户名

    打开Git Bash，键入命令

    如，我的键入git config –global user.name “tangshibao”，回车即可

  2. 配置邮箱

    键入命令

    如：$ git config –global user.email “758109552@qq.com“，回车

  3. 生成密钥（密钥作为无密登录github的凭证，保存在各个电脑上）

    键入命令

    如：ssh-keygen -t rsa -C “758109552@qq.com“
